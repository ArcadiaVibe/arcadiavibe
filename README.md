# ArcadiaVibe

Personal portfolio site built on Azure Static Web Apps with GitHub-based CI/CD.

## Overview

ArcadiaVibe is a live portfolio project focused on building real-world cloud experience while studying for the AZ-104 (Azure Administrator) certification.

This project demonstrates:
- Static website deployment using Azure Static Web Apps
- CI/CD using GitHub Actions
- Source control and versioning with GitHub
- Structured front-end development (HTML/CSS)
